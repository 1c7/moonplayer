MoonPlayer

All-in-One video player that can play videos from Youtube, Bilibili etc. as well as local videos.

Features

MoonPlayer is an interesting player that lets you to enjoy videos. It can play the video online, download it or just open the local videos.

Play videos with an elegant and convenient experience!

Explore videos from your favorite websites!

Play any online videos you want!

Support danmaku too!

Download videos you want!

Full support for hardware decoding!

MoonPlayer is an video player which is

  • Cool
  • Super cool
  • Fuckingly cool
  • Beautiful and lightweight
  • Based on MPV which supports nearly all video formats
  • Focusing on online videos, but also support local videos
  • Hackable through plugins

And there are many more to come.

Installation

Windows / macOS

Download from Release page.

Linux

You can either use the AppImage or install it with Flatpak.

Debian

Add DebianOpt repository, then install with sudo apt install moonplayer.

ArchLinux & Manjaro

Add ArchLinuxCN repository, then install with pacman -S moonplayer.

Development

The source is managed on GitHub. If you encounter any bugs or want to suggest improvements, please use the issue tracker there.
