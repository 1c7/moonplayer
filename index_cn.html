安装

Windows / macOS

从 Release page 下载安装包。

Linux

可以使用 AppImage 或通过 Flatpak 安装。

Debian

添加 DebianOpt 源，然后 sudo apt install moonplayer。

ArchLinux & Manjaro

添加 ArchLinuxCN 源，然后 pacman -S moonplayer。
